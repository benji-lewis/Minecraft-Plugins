# Furniture Plugin

Furniture adds a large catalog of decorative items split into themed modules. Enable the modules you need and customise their item lists in `config.yml`.

## Modules
- Seating
- Tables
- Storage
- Lighting
- Decor
- Outdoor
- Kitchen
- Bedroom
- Bathroom

## Configuration
See `config.yml` for module toggles and furniture lists.

## Documentation
Player documentation lives in `docs/MC-Docs/docs/plugins/furniture.md`.
