# Furniture Plugin

Furniture adds a large catalog of decorative items split into themed modules. Enable the modules you need and customise their item lists in `config.yml`.

## Commands
- `/furniture status` - lists module enablement and loaded module count.
- `/furniture reload` - reloads the configuration file.

## Crafting & Usage
Furniture items are crafted via themed recipes and placed by right-clicking blocks. See the player documentation for full recipes and placement guidance.

## Modules

### Seating
- Config path: `modules.seating`
- Sample items: oak-armchair, spruce-sofa, crimson-bench, leather-recliner

### Tables
- Config path: `modules.tables`
- Sample items: oak-dining-table, birch-coffee-table, granite-side-table

### Storage
- Config path: `modules.storage`
- Sample items: wardrobe, cabinet, dresser, bookshelf

### Lighting
- Config path: `modules.lighting`
- Sample items: chandelier, floor-lamp, wall-sconce

### Decor
- Config path: `modules.decor`
- Sample items: rug, wall-clock, vase

### Outdoor
- Config path: `modules.outdoor`
- Sample items: patio-chair, garden-bench, pergola

### Kitchen
- Config path: `modules.kitchen`
- Sample items: oven-range, kitchen-island, pantry-shelf

### Bedroom
- Config path: `modules.bedroom`
- Sample items: canopy-bed, bunk-bed, bedside-table

### Bathroom
- Config path: `modules.bathroom`
- Sample items: bathtub, sink-basin, towel-rack

## Tuning
- `modules.tuning.max-items-per-module` limits the maximum items loaded per module.

## Configuration
See `config.yml` for module toggles and tuning options.
