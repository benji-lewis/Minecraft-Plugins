# KimJongUnMob

Adds a themed FancyNPC styled with a dark suit and custom name. Interacting with the NPC yields thermonuclear missile parts that can be crafted into a thermonuclear missile item, and the NPC can be fought like a normal mob.

## Features
- **Kim Jong Un themed NPC**: A FancyNPC with a dark suit aesthetic and custom skin.
- **Missile parts**: Nose cone, body, fins, and warhead core drop on NPC interaction without despawning.
- **Thermonuclear missile**: Craft the parts into a missile that launches when placed on a powered block.

- `kimjongunmob.spawn` (default: op)

## Crafting Recipe
```
 N 
BWB
 F 
```
- **N** = Thermonuclear Nose Cone
- **B** = Thermonuclear Missile Body
- **W** = Thermonuclear Warhead Core
- **F** = Thermonuclear Missile Fins

## Usage
1. Interact with the themed NPC to collect all four missile parts (the NPC stays active).
2. Craft the **Thermonuclear missile** using the recipe above.
3. Right-click a block that is receiving redstone power to launch the missile.

## Notes
- FancyNPCs is required to spawn the Kim Jong Un NPC.
- The missile launches upward and detonates after a short fuse.
