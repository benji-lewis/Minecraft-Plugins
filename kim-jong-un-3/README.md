# Kim Jong Un 3 (Nova Addon)

Kim Jong Un 3 is a Nova addon for PaperMC that spawns a themed mob which drops randomized missile or launchpad parts. Players can assemble the parts into a launchpad and missile, then trigger a dramatic missile launch. Items are registered as Nova items so they integrate with Nova's resource pack pipeline.
